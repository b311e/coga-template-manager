# Sheet Template Documentation

## Overview
JBC Excel worksheet template for default Sheet.xltx in XLSTART folder.

## Template Details
- **Type**: Excel Worksheet Template (.xltx)
- **Purpose**: Default template when users insert new worksheets
- **Deployment**: XLSTART folder as Sheet.xltx
- **Theme**: JBC Office theme with standardized colors and fonts
